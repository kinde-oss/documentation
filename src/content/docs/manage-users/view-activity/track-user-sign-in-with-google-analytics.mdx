<Aside type="upgrade">

Google Analytics is available on [Kinde Pro plan or higher](https://kinde.com/pricing/).

</Aside>

Use Google Analytics to track user sign-in events in Kinde. No activity in Kinde is tracked, but you will be able to see where a user’s journey ended in signing in or signing up via our platform.

## Get your Google measurement ID (G-XXXX)

1. Sign in to your Google Analytics account
2. Go to **Admin > Data collection and modifications > Data Streams**
3. Select the data stream you want (e.g., Website), the details pane opens
4. The `G-XXXX` code appears under the `MEASUREMENT ID` field. Copy the code

## Set up Google Analytics in Kinde

1. In Kinde, go to **Settings > Environment > Environment Details**
2. Scroll to the **Third-party analytics > Google Analytics** and enter your `G-XXXX` code in the **Google Analytics** field
3. Select **Save**

## Google tag manager not supported

As of now, Google Tag Manager is not supported in Kinde due to security concerns. Learn more about our [security principles](/trust-center/security/security-at-kinde/).
